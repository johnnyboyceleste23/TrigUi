# TrigUi

A simple Python GUI application for solving and working with trigonometry problems.

## Overview
TrigUi is a beginner-friendly desktop application built in Python.  
The goal of this project is to provide an easy interface for performing trigonometric calculations and practicing Python GUI development.

This project was created as part of learning:
- Python programming
- GUI development
- Git & GitHub workflow

## Features
Current features:
- User input for trigonometric calculations
- Graphical user interface
- Real-time result display

## Technologies Used
- Python 3
- Tkinter (GUI framework)
- Git & GitHub
